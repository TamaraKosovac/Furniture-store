# CozyHaven - salon namještaja

**CozyHaven** je aplikacija namijenjena zaposlenima u salonu namještaja. Omogućava jednostavnu manipulaciju zaposlenima, proizvodima, izdavanjima računa, te samim računima. Dodatno, aplikacija podržava promjenu jezika i teme.


## Sadržaj
- Početna stranica
- Administratorska stranica
- Stranica za zaposlene


## Početna stranica 
Početna stranica sadrži login formu sa poljima za unos korisničkog imena i lozinke. Na ovaj način je omogućena prijava kako administratora tako i zaposlenih.
